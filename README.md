# Laravel Dusk Mocking

This package is a fork from the [original](https://github.com/NoelDeMartin/laravel-dusk-mocking), which is no longer maintained. This version makes a few tweaks and is mostly intended for internal use, but feel free to submit a PR if you have something useful to contribute.

The original readme can be found [here](https://github.com/NoelDeMartin/laravel-dusk-mocking/blob/master/README.md).
